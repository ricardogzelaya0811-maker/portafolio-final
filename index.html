Sobre Mí

Soy Ricardo Zelaya, un estudiante comprometido con el dominio de las herramientas de productividad Microsoft Office, actualmente en preparación para obtener la certificación Microsoft Office Specialist (MOS). Mi interés por la tecnología me ha llevado a explorar no solo el uso estándar de las aplicaciones, sino también su integración con inteligencia artificial para optimizar procesos y elevar la calidad del trabajo profesional.

A lo largo de este portafolio presento una selección de documentos que evidencian mi competencia en el manejo avanzado de Microsoft Office. Cada proyecto fue desarrollado con atención al detalle y apoyo de herramientas de IA, reflejando mi capacidad para combinar habilidades técnicas con pensamiento analítico y creatividad orientada a resultados.

Proceso creativo

Cómo Usé la IA

Durante el desarrollo de este portafolio utilicé Claude (IA de Anthropic) en cada etapa: ideación, redacción de documentos, fórmulas Excel y diseño completo de este sitio web.

🧠

Ideación

Pedí sugerencias de documentos Word y Excel que demostraran habilidades MOS avanzadas para un portafolio coherente.

✍️

Redacción

Claude generó textos formales, corrigió estilo y ortografía, y sugirió mejoras en todos mis documentos Word.

📊

Fórmulas Excel

La IA proporcionó fórmulas exactas (BUSCARV, SUMAR.SI, SI anidado) con explicaciones paso a paso.

🎨

Diseño Web

Claude generó todo el HTML, CSS y JavaScript con diseño UX/UI profesional, login, modo oscuro y responsive.
