Sobre Mí

Soy Ricardo Zelaya, un estudiante comprometido con el dominio de las herramientas de productividad Microsoft Office, actualmente en preparación para obtener la certificación Microsoft Office Specialist (MOS). Mi interés por la tecnología me ha llevado a explorar no solo el uso estándar de las aplicaciones, sino también su integración con inteligencia artificial para optimizar procesos y elevar la calidad del trabajo profesional.

A lo largo de este portafolio presento una selección de documentos que evidencian mi competencia en el manejo avanzado de Microsoft Office. Cada proyecto fue desarrollado con atención al detalle y apoyo de herramientas de IA, reflejando mi capacidad para combinar habilidades técnicas con pensamiento analítico y creatividad orientada a resultados.

Cómo Usé la IA

Durante el desarrollo de este portafolio utilicé Claude (IA de Anthropic) como asistente en cada etapa: ideación, redacción de documentos, generación de fórmulas Excel y diseño de este sitio web.

🧠

Ideación de documentos

Pedí a la IA sugerencias de documentos Word y Excel que demostraran habilidades MOS avanzadas, para estructurar un portafolio coherente y profesional.

✍️

Redacción y corrección

Usé prompts específicos para que Claude generara texto formal, corrigiera ortografía y estilo, y sugiriera mejoras en todos mis documentos Word.

📊

Fórmulas Excel

Describí mis necesidades y la IA me proporcionó fórmulas exactas (BUSCARV, SUMAR.SI, SI anidado), explicando cada una paso a paso.

🎨

Diseño del portafolio

Proporcioné especificaciones detalladas y Claude generó todo el HTML, CSS y JavaScript con diseño UX/UI profesional y responsive.

📋 Prompts reales que usé con Claude

Para documentos Word
"Actúa como experto en Microsoft Word. Crea una carta formal de presentación para el sector tecnológico con encabezado profesional, 3 párrafos bien estructurados y cierre formal en español."
Para hojas Excel
"Crea una plantilla de control de inventario en Excel con: Código, Producto, Categoría, Cantidad, Precio, Total. Incluye fórmulas automáticas y formato condicional para alertar stock mínimo."
Para este portafolio
"Actúa como experto senior en desarrollo web frontend. Crea mi portafolio profesional MOS con diseño navy y dorado, carga dinámica desde JSON, GitHub Pages compatible, responsive y con foto de avatar."
